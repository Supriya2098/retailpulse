
# 🚀 RetailPulse Deployment Guide

## Streamlit Community Cloud Deployment

RetailPulse is deployed using Streamlit Community Cloud.

### Benefits

- Free hosting
- HTTPS-enabled public URL
- Auto-deployment on every push to main
- Easy GitHub integration
- Simple rollback and redeploy
